Configuration pour déployer le modèle Phi-3.5-Medical avec Triton Inference Server.

### Option 1: Ollama (Recommandé - Simple)
```bash
ollama pull microsoft/phi-3.5-mini-instruct
ollama serve
```

### Option 2: Triton (Avancé - Production)
Configuration fournie ci-dessous.

Voir exemple dans `/scripts`
